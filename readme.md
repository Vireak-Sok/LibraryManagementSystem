# Library Management System

***

## Prerequisites
> - Node.js
> Android
> - Java 8+ JDK
> - Android SDK
> iOS
> - Xcode
> - Cocoapods
***

## Project Setup
```
npm install
```

## Install Pod Dependencies (For MacOS Only)
```
cd ios && pod install && cd ..
```

### Build & Run
```
npm run android
npm run ios
```
